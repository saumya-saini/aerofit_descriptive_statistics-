# Aerofit Treadmill Data Analysis

##  Overview

Analyzed Aerofit customer data to understand **purchase behavior**, identify **key customer segments**, and generate **actionable business insights** for different treadmill products.


##  Dataset

The dataset includes customer attributes such as:

* Product (KP281, KP481, KP781)
* Age, Gender, Marital Status
* Income, Usage, Fitness, Miles


##  Objective

* Understand **who buys which treadmill**
* Identify **factors influencing product choice**
* Build **customer profiles for each product**
* Use **probability analysis** for deeper insights


##  Key Analysis

* Exploratory Data Analysis (EDA)
* Univariate & Bivariate Analysis
* Correlation Analysis
* Contingency Tables
* Marginal, Conditional & Joint Probabilities


##  Key Insights

* **KP281** is the most popular treadmill, preferred by a broad customer base.
* **KP781 (premium model)** is primarily purchased by:

  * High-income customers
  * High fitness individuals
  * High-usage users
* Male customers show a higher tendency to purchase **premium treadmills**.
* Customers with higher fitness levels tend to run more miles and prefer advanced models.


##  Customer Segments

* **KP281 (Entry-Level):** Beginner users, moderate income, balanced gender distribution
* **KP481 (Mid-Level):** Moderate fitness users, regular usage
* **KP781 (Premium):** High-income, high-fitness, heavy users (predominantly male)


##  Tools Used

* Python (Pandas, NumPy)
* Matplotlib, Seaborn


##  Business Impact

* Enables **targeted marketing strategies**
* Helps identify **high-value customer segments**
* Supports **product positioning and upselling strategies**


##  Project Highlights

* Applied **probability concepts (marginal, conditional, joint)** in real-world analysis
* Derived **data-driven customer profiles**
